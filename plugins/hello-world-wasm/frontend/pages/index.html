Hello World Plugin (WASM)

A sample Rust/WASM plugin demonstrating hooks, routes, host functions, and frontend panels

WASM Backend
Compiled from Rust to wasm32-wasip1 via Extism PDK. Routes served at /api/ext/hello-world-wasm/.
Entity Hooks
Implements on_entity_save, on_entity_create, on_entity_delete, and on_entity_validate hooks.
Sidebar Panel
A "Plugin Notes" panel appears in every entity editor sidebar, letting you add notes per entity.
Entity Tab
An "Entity Stats" tab auto-injects into entity editors showing view counts and activity.
Host Functions
Calls host-provided entity queries and settings via Extism vars, matching the WIT interface contract.
Plugin Page
This page! Full HTML pages served through sandboxed iframes with studio navigation.
