Hello World Plugin

A sample plugin demonstrating the City of Brains plugin system capabilities

Backend Routes
Custom API endpoints at /api/ext/hello-world/ with health checks and greeting messages.
Event Handlers
Listens for entity.* events on the backend event bus and logs all entity lifecycle activity.
Sidebar Panel
A "Plugin Notes" panel appears in every entity editor sidebar, letting you add notes per entity.
Entity Tab
An "Entity Stats" tab auto-injects into entity editors showing view counts and activity.
Plugin Settings
Configurable greeting message and event logging toggle via Settings > Plugins.
Plugin Page
This page! Full HTML pages served through sandboxed iframes with studio navigation.
