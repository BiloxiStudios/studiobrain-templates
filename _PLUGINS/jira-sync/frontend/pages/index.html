Setup Instructions
  1. Go to Settings > Plugins > Jira Sync in the Studio sidebar.
  2. Enter your Atlassian Cloud URL, e.g. https://yourstudio.atlassian.net
  3. Enter the email associated with your Atlassian account.
  4. Generate an API token at id.atlassian.com/manage-profile/security/api-tokens and paste it.
  5. Set the Default Project Key (e.g. SBAI, PROD).
  6. Optionally configure Entity-to-Issue Type Mapping and Status Mapping as JSON.
  7. Enable Auto-create and/or Auto-transition toggles as desired.
